# Iridescence 组件明亮配色方案

## 🎨 新的明亮活泼配色

为了创造更加明亮、活泼的视觉效果，我们重新设计了 Iridescence 背景的配色方案。

## 🌈 主要配色

### 1. HeroSection - 长春花蓝 (Periwinkle Bright)
- **颜色**: `#a59eff` (RGB: 165, 158, 255)
- **效果**: 明亮的长春花蓝，给人科技感和未来感
- **参数**: 
  - `amplitude: 0.15` - 增强的波动效果
  - `speed: 1.0` - 更快的动画速度
  - `mouseReact: true` - 鼠标交互增强

### 2. SmartCTASection - 活力薄荷绿 (Mint Vibrant)
- **颜色**: `#ccfaf0` (RGB: 204, 250, 240)
- **效果**: 清新的薄荷绿，给人活力和希望感
- **参数**:
  - `amplitude: 0.10` - 适中的波动效果
  - `speed: 0.8` - 舒缓的动画速度
  - `opacity: 0.3` - 30% 透明度，不干扰内容

## 🎯 配色原则

### 明亮化策略
1. **提升饱和度**: 从深色调转向中等饱和度
2. **增加亮度**: 选择更亮的颜色值
3. **保持品牌一致性**: 基于 She Sharp 品牌色彩系统
4. **创造层次感**: 不同区域使用不同的明亮色彩

### 色彩心理学
- **长春花蓝**: 科技、创新、信任
- **薄荷绿**: 活力、希望、成长
- **整体效果**: 年轻、现代、充满活力

## 🔧 技术实现

### 颜色值转换
```typescript
// 从 HEX 到 RGB (0-1 范围)
periwinkleBright: [0.647, 0.620, 1.0]    // #a59eff
mintVibrant: [0.800, 0.980, 0.941]       // #ccfaf0
```

### 参数优化
- **HeroSection**: 更高的 amplitude 和 speed，创造震撼开场
- **SmartCTASection**: 适中的参数，保持优雅的交互体验

## 📱 响应式考虑
