# She Sharp 前端开发计划

## 1. 项目概述

### 组织背景
- **组织名称**: She Sharp
- **性质**: 新西兰注册慈善机构 (#CC57025)
- **使命**: 通过赋能女性在 STEM 领域的发展，缩小性别差距
- **成立时间**: 2014年
- **创始人**: Dr. Mahsa McCauley

### 网站目标
1. 展示组织使命和价值观
2. 提供活动信息和报名入口
3. 展示导师计划和成功案例
4. 吸引赞助商和捐赠
5. 建立社区连接和会员网络

## 2. 网站架构

### 主导航结构
```
├── 首页 (Home)
```

## 3. 设计系统

### 视觉风格
- **主色调**: 
  - 紫色系（体现科技感和女性力量）
  - 辅助色：青色、粉色渐变
- **字体**: 
  - 标题：现代无衬线字体
  - 正文：易读的系统字体
- **图像风格**: 
  - 真实的活动照片
  - 多元化的女性形象
  - 科技感的图标和插画

### UI组件库
基于 shadcn/ui 定制：

1. **布局组件**
   - Header (带透明/固定效果)
   - Footer
   - Container
   - Section
   - Grid System

2. **导航组件**
   - MainNav (响应式)
   - MobileMenu
   - Breadcrumb
   - TabNav

3. **内容组件**
   - HeroSection
   - TeamMemberCard
   - EventCard
   - TestimonialCard
   - SponsorLogo
   - StatisticCounter
   - CTASection

4. **交互组件**
   - DonateButton
   - NewsletterForm
   - ContactForm
   - EventRegistration
   - VideoPlayer

5. **媒体组件**
   - ImageGallery
   - VideoEmbed
   - PodcastPlayer

## 4. 技术架构

### 技术栈
- **框架**: Next.js 15 (App Router)
- **样式**: Tailwind CSS v4
- **组件库**: shadcn/ui
- **动画**: Framer Motion
- **图标**: Lucide Icons
- **表单**: React Hook Form + Zod
- **状态管理**: Zustand (如需要)
- **媒体优化**: next/image, next/video

### 项目结构
```
/app
  /(site)           # 公共网站路由
    /page.tsx       # 首页
    /about
    /events
    /mentorship
    /media
    /contact
    /donate
    /join
  /api              # API路由（未来扩展）
/components
  /ui               # shadcn/ui组件
  /sections         # 页面section组件
  /layout           # 布局组件
  /shared           # 共享组件
/lib
  /utils            # 工具函数
  /constants        # 常量定义
/public
  /images
  /videos
/content            # Markdown内容
```

## 5. 开发阶段

### 第一阶段：基础架构（第1-2周）
1. ✅ 清理现有模板代码
2. 设置项目结构
3. 配置设计系统（颜色、字体、间距）
4. 创建基础布局组件（Header、Footer）
5. 实现响应式导航

### 第二阶段：核心页面（第3-4周）
1. 首页开发
   - Hero section with video
   - 统计数据展示
   - 赞助商展示
   - CTA sections
2. 关于我们页面
   - 团队成员展示
   - 组织历史时间线
3. 联系页面
   - 联系表单
   - 地图集成

### 第三阶段：活动模块（第5-6周）
1. 活动列表页
   - 过滤和搜索功能
   - 活动卡片组件
2. 活动详情页
   - 活动信息展示
   - 报名功能（初期可链接外部）
3. Google教育者大会专题页

### 第四阶段：导师计划模块（第7-8周）
1. 导师计划概览页
   - 计划介绍
   - 成功数据展示
2. 导师展示页
   - 导师卡片网格
   - 筛选功能
3. 申请成为学员页

### 第五阶段：媒体中心（第9-10周）
1. 播客页面
2. 新闻简报订阅
3. 新闻报道聚合
4. 图片集（画廊功能）
5. 影响力报告下载

### 第六阶段：优化和完善（第11-12周）
1. 性能优化
   - 图片懒加载
   - 代码分割
   - SEO优化
2. 可访问性改进
3. 动画和过渡效果
4. 表单验证和用户反馈
5. 404和错误页面
