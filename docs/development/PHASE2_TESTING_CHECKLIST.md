### 测试账户
如需创建新测试账户，请按以下流程：

2. **Mentee测试账户**
   - 注册新账户
   - 在Welcome页面选择 "Find a Mentor"
   - 完成onboarding流程
   - 填写学员档案信息

## ✅ 功能测试清单

### 1. 导师功能测试 (Mentor Account)

- [ ] 登录mentor账户

- [ ] 点击"Edit Profile"进入 `/dashboard/mentor-profile`
- [ ] 测试以下功能：
  - [ ] 填写/修改专业信息（职位、公司、经验年限）
  - [ ] 选择专业领域（至少选择3个）
  - [ ] 设置可用时间（每月小时数）
